# vue-geo-example
Demo app powered by [vue-d3-geo](https://github.com/komushi/vue-d3-geo/)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
